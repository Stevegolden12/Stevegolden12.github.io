URL Shortener Microservice

How to use this site:

  • In the box below type in the web address you like to be shortened. Depening on the url 3 things will happen:
    1. If the url is correct and has not already have a urlshortener already then you will get the following message:
      'The (url) can be access by using [this_project_url]/number/(shorturl)'.
    2. If the url is correct and has a urlshortener already then you will get the following message:
      'The (url) is in the database and can be access by using [this_project_url]/number/(shorturl)'
    3. If the url is not valid you will get the following message:
      '{ "error": "invalid URL" }'
  • The shortened url can be accesed by using [this_project_url]/number/(shorturl)
